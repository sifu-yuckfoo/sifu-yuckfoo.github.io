PURPOSE
------------------------------------------------------------------------------
This manifest defines the first public-safe research/publications set for the
Akamai Intelligence Group website.

The publication posture is intentionally curated. AIG should publish the papers
that establish credibility, technical framing, and evaluation discipline while
holding back operationally sensitive implementation details, private concepts,
and anything not ready for external review.

ATTRIBUTION STANDARD
------------------------------------------------------------------------------
Public-facing papers should use this format:

  Author: Christopher Ramos
  Research and drafting support: Deus ex Machina

Internal working papers may continue using:

  Christopher Ramos + Deus ex Machina

Rationale:

  - Christopher Ramos is the accountable human author and principal.
  - Deus ex Machina is acknowledged honestly as AI-assisted drafting and
    technical synthesis support.
  - The work remains credible to reviewers, funders, partners, and agencies.

PUBLICATION TIERS
------------------------------------------------------------------------------
Tier 1 -- public-safe flagship papers:
  Strong enough for the website now. These should be externally readable and
  should support DICE, DARPA, partner, and technical credibility conversations.

Tier 2 -- capability briefs / whitepapers:
  Useful for selective sharing, but may need polishing, sanitization, or tighter
  external framing before prominent website placement.

Tier 3 -- restricted / internal:
  Keep off the public website until explicitly cleared. This includes private
  project material, operationally sensitive implementation details, raw strategy,
  sensitive cyber mechanics, and Oasis-related concepts.
